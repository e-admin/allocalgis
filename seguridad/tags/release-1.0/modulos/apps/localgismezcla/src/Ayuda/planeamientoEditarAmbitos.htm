Editor de Ambitos de Gestión del Suelo

 

Cuando se selecciona esta opción, se abre la pantalla de edición ámbitos de gestión del suelo, donde el usuario puede visualizar o realizar cambios en los Ámbitos de Gestión. Esta opción sólo estará habilitada para el usuario con el permiso de "Editor de Ambitos de Gestion", en caso contrario no se podrá acceder a esta pantalla.

 

Para esta pantalla, el usuario dispone de un mapa preconfigurado con las capas de Gestión del Suelo, unas capas básicas de referencia y de un conjunto de herramientas de edición.
