Reparto

     Con esta opción el usuario podrá realizar el reparto de cultivos ó locales comunes entre los diferentes destinos.

     Para ello se debe seleccionar el cultivo ó local común. En su pantalla de edición aparece un botón Reparto que seleccionamos.

     Ver Cultivos Comunes

     Ver Locales Comunes

     Una vez seleccionada la opción, la aplicación lanza una ventana con toda la información específica a rellenar para el reparto del elemento. Dicha información es la siguiente:

  • Datos del reparto:
    • Parcela catastral1 y parcela catastral2: indican la referencia catastral de la finca a la que pertenece dicho local.
    • Nº de Orden: Número de orden del elemento de construcción en caso de que el elemento a repartir sea un local, ó el número de orden del cargo en caso de repartir un cultivo.
  • Modalidad del reparto:
    • Forma de reparto: Indica el tipo de reparto, a cargos ó a locales.
    • Porcentaje de reparto: porcentaje del reparto del cultivo.

     Botón Aceptar

     Cuando el usuario haya finalizado de rellenar la información pertinente al reparto, deberá pulsar este botón para que los cambios realizados tengan efecto. Si alguno de los campos introducidos no es correcto saltará un mensaje de error con el problema ocurrido para poder subsanarlo. Hasta que la información recogida en esta pantalla no sea correcta el reparto no se realizará.

     Botón Cancelar

      Si queremos anular los cambios introducidos en la pantalla y que se matenga la información anterior, el usuario debe pulsar este botón Cancelar.
